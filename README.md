# Python YT to MP3/MP4 Script

This is a simple desktop app that helps you download YouTube content in the format you actually want.

Instead of dealing with command-line tools, you just open the app, paste a YouTube link, choose `MP3` for audio or `MP4` for video, pick where to save it, and click download.

The goal of this project is to keep things quick and beginner-friendly: a clean interface, minimal steps, and no extra technical setup during normal use.
